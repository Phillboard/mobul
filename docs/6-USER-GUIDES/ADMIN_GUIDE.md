# Admin Guide

## Overview

Comprehensive guide for platform administrators managing organizations, clients, users, and system configuration.

---

## User Management

### Creating Users

1. Navigate to **Administration** → **User Management**
2. Click **Invite User**
3. Enter email and select role
4. Choose organization/client assignment
5. Send invitation

### User Roles

- **Admin** - Full platform access
- **Tech Support** - Support and troubleshooting
- **Agency Owner** - Multi-client management
- **Company Owner** - Single client management
- **Developer** - API and integration access
- **Call Center** - Redemption operations only

---

## Organization Management

### Creating Organizations

1. Navigate to **Administration** → **Organizations**
2. Click **Create Organization**
3. Configure org type (Internal, Agency, Direct)
4. Assign admin users

---

## System Health

Monitor platform performance:

- **Analytics** - Usage metrics
- **Performance** - Response times
- **Error Tracking** - System errors
- **Alerts** - Critical notifications

---

## Best Practices

1. **Assign minimal roles** - Principle of least privilege
2. **Monitor audit logs** - Track admin actions
3. **Review permissions** - Quarterly access review
4. **Enable 2FA** - Require for admin accounts
5. **Document changes** - Keep change log
