## Best Practices

1. **Clean contact data** - Valid emails/phones
2. **Test small** - Pilot before scaling
3. **Monitor daily** - Check metrics regularly
4. **Follow up quickly** - Contact leads within 24 hours
5. **Measure ROI** - Track cost per acquisition
