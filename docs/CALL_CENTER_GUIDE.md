# Call Center Gift Card Redemption Guide

## Quick Reference

### How to Redeem a Gift Card

1. **Ask Customer for Code**
   - Request the redemption code from the customer
   - Codes are in format: `ABC-1234` (letters-numbers)

2. **Enter Code**
   - Type or paste the code into the input field
   - Code is automatically converted to uppercase
   - Click "Look Up" or press Enter

3. **Verify Customer**
   - Review customer details shown on screen
   - Confirm name, phone, or email with the customer
   - Check campaign/audience information

4. **Card Provisioned**
   - Gift card details appear automatically
   - Use "Copy All" button to copy all details at once
   - Read card information to customer:
     - Brand name and value
     - Gift card code
     - Card number (if applicable)
     - Expiration date

5. **Send Card Details (Optional)**
   - If customer has a phone number, click "Send via SMS"
   - Card details will be texted to their phone
   - Customer can also add to Apple/Google Wallet

6. **Start Next Redemption**
   - Click "Start New Redemption" button
   - Form resets for next customer

## Troubleshooting

### Code Not Working
1. Verify code format (ABC-1234)
2. Check for spaces or special characters
3. Confirm code hasn't been redeemed
4. Check if code requires approval

### SMS Not Sending
1. Verify customer has a valid phone number
2. Check network connectivity
3. Try sending again after 30 seconds
4. If repeated failures, escalate to tech support

### System Running Slow
1. Refresh your browser
2. Clear browser cache
3. Check internet connection
4. Contact tech support if issue persists

### Can't Access System
1. Verify you're logged in
2. Check your user role (must be call_center or admin)
3. Clear cookies and try again
4. Contact tech support

## Escalation Procedures

### When to Escalate to Supervisor:
- Pool is empty (no cards available)
- Code requires approval
- Customer disputes rejection
- Technical error persists after retry
- Customer requests refund or alternative
- System showing unexpected errors

### When to Escalate to Tech Support:
- System completely unresponsive
- Multiple SMS failures
- Login issues
- Performance issues after browser refresh
- Data not loading correctly

### How to Escalate:
1. Place customer on hold (don't disconnect)
2. Note customer details and issue
3. Contact supervisor via internal chat/phone
4. Provide redemption code and error message
5. Wait for supervisor response before returning to customer

## Security & Compliance

### Data Protection:
- Never share customer data via email or text
- Don't write down codes on paper
- Clear your screen between customers
- Log out when leaving workstation
- Report suspicious activity immediately

### Privacy:
- Only access customer data when needed
- Don't discuss customer information in public areas
- Verify customer identity before sharing details
- Follow company privacy policy

### Fraud Prevention:
- Watch for customers requesting multiple redemptions
- Report patterns of rejected codes
- Alert supervisor if same customer calls repeatedly
- Don't override system rejections

## Frequently Asked Questions

**Q: What if customer lost their code?**  
A: Customer must contact the company that issued the code. We cannot look up codes without the code itself.

**Q: Can I manually enter a gift card?**  
A: No. All cards must be provisioned through the system for tracking and compliance.

**Q: What if customer wants a different brand?**  
A: Cards are assigned automatically from the available pool. We cannot swap brands.

**Q: How long are codes valid?**  
A: Varies by campaign. Check the campaign information or ask supervisor.

**Q: Can customer use card immediately?**  
A: Yes! Cards are active as soon as provisioned. Customer can use online or in-store.

**Q: What if customer says card isn't working at store?**  
A: Transfer to customer service. This is outside call center scope.
