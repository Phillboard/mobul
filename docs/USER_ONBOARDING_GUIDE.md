# User Onboarding Guide

## Overview
This guide outlines the onboarding experience for new users of the platform.

## Onboarding Checklist
New users are presented with a 6-step checklist to guide them through initial setup:

### 1. Complete Your Profile
- Add display name
- Upload profile picture
- Set preferences
- **Status Check**: Profile has display_name populated

### 2. Set Up Your Organization
- Add company name and industry
- Configure brand colors and logo
- Set timezone
- **Status Check**: At least one client record exists

### 3. Create Your First Template
- Choose from starter templates
- Use AI to generate design
- Customize in template builder
- **Status Check**: At least one template record exists

### 4. Upload an Audience
- Download sample CSV
- Upload recipient list
- Review data hygiene results
- **Status Check**: At least one audience record exists

### 5. Create a Campaign
- Follow campaign creation wizard
- Select template and audience
- Configure landing page
- Set up gift card rewards
- **Status Check**: At least one campaign record exists

### 6. Build a Landing Page
- Use AI to generate page
- Customize content and design
- Publish page
- **Status Check**: At least one landing_page record exists

## Progress Tracking
- Progress is calculated automatically based on database records
- Users can refresh progress at any time
- Completion is celebrated with confetti animation
- Progress bar shows percentage complete

## First Campaign Walkthrough
New users are guided through creating their first campaign:

1. **Template Selection**
   - Choose from industry templates
   - See preview before selecting
   - Option to start from scratch or use AI

2. **Audience Setup**
   - Upload CSV with guided field mapping
   - View data quality report
   - Fix issues before proceeding

3. **Landing Page**
   - AI generates personalized page
   - Edit content inline
   - Preview before publish

4. **Gift Card Configuration**
   - Select pool or create new
   - Set redemption conditions
   - Configure SMS templates

5. **Review & Launch**
   - See campaign summary
   - Verify all settings
   - Launch or save as draft

## Common Onboarding Issues

### CSV Upload Errors
**Issue**: Invalid file format or missing columns
**Solution**: Provide sample CSV download, show field mapping UI

### Gift Card Confusion
**Issue**: Users don't understand pools vs individual cards
**Solution**: Add explainer tooltip, link to video tutorial

### Landing Page Customization
**Issue**: Users struggle with editor
**Solution**: Provide templates, AI assistance, and video tutorial

### Campaign Status Confusion
**Issue**: Users don't know why campaign is "pending"
**Solution**: Show clear status indicators with explanations
