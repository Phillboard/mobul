# Call Center Guide

## Overview

Operational guide for call center agents handling inbound calls and gift card redemptions.

---

## Call Workflow

### 1. Answer Call
- Greet caller professionally
- Ask for confirmation code

### 2. Look Up Recipient
- Enter 8-character code
- Verify caller identity
- Review campaign details

### 3. Collect Information
- Confirm/collect email or phone
- Verify mailing address
- Note any special requests

### 4. Select Condition
- Review campaign conditions
- Confirm which was met
- Select from dropdown

### 5. Deliver Gift Card
- Choose delivery method (SMS/Email)
- Confirm contact information
- Send gift card code
- Provide redemption instructions

---

## Common Issues

**Code Not Found:**
- Verify code entry (no spaces)
- Check if uppercase/lowercase matters
- Confirm campaign is active

**No Cards Available:**
- Alert supervisor immediately
- Promise delivery within 24 hours
- Log pending reward

**Caller Can't Redeem:**
- Walk through redemption steps
- Offer alternative delivery method
- Escalate to supervisor if needed

---

## Best Practices

1. **Be professional** - Friendly and helpful tone
2. **Verify identity** - Protect against fraud
3. **Document thoroughly** - Add notes to call session
4. **Follow scripts** - Consistent experience
5. **Escalate issues** - Don't guess, ask for help
